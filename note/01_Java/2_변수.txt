[ 2 ] 자바에서의 변수
1. 변수(variable)란?
(1) 프로그램 작업을 처리하기 위해 하나의 값을 저장할 수 있는 메모리 공간을 말한다.
(2) 임의의 메모리 공간에 이름을 붙여 관리하는 것이다.
(3) 자바에서는 다양한 타입을 저장할 수 없고, 한 가지 타입만 값으로 저장될 수 있다.
(4) 식별자(identifier)라고도 한다 : 자바 코드에서 변수로 입력시킨 이름을 의미한다.
2. 기본 자료형
자료형  	키워드  	크기  	표현 범위
논리형 	boolean  1byte 	true, false
문자형	char	2byte	0~65,535
정수형	int	4byte	-2,147,483,648 ~ 2,147,483,647
실수형	double	8byte	1.7E308 ~ + 1.7E308
3.기본 데이터 type과 참도 데이터 type의 이해
(1) 기본 데이터 type (primitive 기본 자료형)
 - 메모리에 있는 실제 값 = 변수 데이터 값
(2) 참조 데이터 type(Object, 객체 자료형)
메모리에 있는 실제 값(stack 영역) = 변수가 저장된 주소(실제 변수 데이터는 heap 영역에 저장)
